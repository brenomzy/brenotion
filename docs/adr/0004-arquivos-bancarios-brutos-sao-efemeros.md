# Arquivos bancários brutos são efêmeros

Extratos e faturas usados na importação serão mantidos somente durante extração e validação, sendo apagados depois da confirmação ou rejeição do lote; permanecem apenas dados estruturados, hash e metadados de auditoria. O Cofre Fiscal retém documentos fiscais deliberadamente, mas não funciona como arquivo permanente de dados bancários brutos, reduzindo o impacto de uma exposição sem impedir reimportação a partir dos originais mantidos pelo Titular.
