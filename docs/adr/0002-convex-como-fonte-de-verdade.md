# Convex como fonte de verdade em nuvem

Convex será a fonte de verdade para dados estruturados, jobs, sincronização, auditoria e documentos, enquanto o Android manterá apenas cache e credenciais protegidas. A escolha aceita dependência de um backend gerenciado em troca de webhooks, acesso web, processamento central, backup e continuidade entre dispositivos; regras do domínio permanecem isoladas para reduzir o custo de uma migração futura.
