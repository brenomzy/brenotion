# Aplicação universal Expo com Android primeiro

O produto usará uma única aplicação Expo Router para Android e web, com Android como experiência principal e a web inicialmente limitada a upload e cofre documental. Isso evita dois frontends antes de existir necessidade real, preserva a possibilidade de componentes específicos por plataforma e permite que o companion web cresça gradualmente; um frontend separado só será considerado quando a experiência desktop exigir outra arquitetura.
