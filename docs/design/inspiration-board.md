# Board de inspirações

Pesquisa realizada no Mobbin em 14 de julho de 2026. Os links canônicos preservam a origem; imagens de terceiros não são copiadas para o repositório. As referências informam padrões de interação, não identidade visual.

## Critério de seleção

- ajuda a explicar uma decisão financeira ou estado operacional;
- pode ser adaptada aos conceitos canônicos do Brenotion;
- não depende de movimentação automática de dinheiro;
- mantém utilidade quando os dados estão incompletos ou desatualizados;
- evita transformar patrimônio, saldo ou investimento em sinônimo de disponibilidade.

## Hipóteses abertas

- A tela inicial deve usar uma única grande superfície para o Disponível para Gastar ou integrar o valor ao fluxo de próxima ação?
- A revisão semanal funciona melhor como uma fila contínua ou como quatro etapas curtas?
- A separação Empresa/Pessoal deve aparecer como contexto persistente, segmentação local ou ambos?
- Meses de autonomia comunicam melhor a Reserva Familiar do que uma barra percentual em todos os marcos?
- O companion web precisa de navegação lateral no primeiro perímetro ou duas rotas diretas são suficientes?
